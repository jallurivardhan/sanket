# Sanket
### The signal between financial minds

> Google Cloud Rapid Agent Hackathon 2026 
> — Financial Services Track | Arize Partner

**What it does:** Sanket is a cross-institutional AI agent 
signal network. When one bank's AI detects fraud, every 
connected bank knows within 500ms — with zero raw customer 
data shared.

**Tech stack:** Gemini 3.1 Pro · LangGraph · Google Cloud 
Agent Builder · Arize Phoenix · FastAPI · Redis · React

**Quick start:**
- cp .env.example .env
- docker-compose up
- open http://localhost:8000/health
